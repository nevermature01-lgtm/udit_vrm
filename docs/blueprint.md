# **App Name**: Eternal Hearts

## Core Features:

- Romantic Intro: Full-screen intro with animated text and a 'Start Our Journey' button to smoothly scroll to the timeline.
- Love Timeline: Animated vertical timeline displaying key moments with glowing dots, descriptions, and heart icons. Utilizes slide-in animations and glassmorphism effect.
- Memory Gallery: Grid layout gallery to showcase shared memories. Clicking opens a modal with larger image, caption, and description.
- Love Notes Cards: Interactive flip cards displaying reasons 'Why I Love You' with a smooth 3D flip effect on hover.
- Special Message: Animated typewriter effect to reveal a heartfelt message with a soft glowing effect.
- "Will You Be My Valentine?" Section: Interactive section with an animated question, 'YES' and a mischievous, moving 'NO' button.  Full-screen heart explosion on 'YES' click.
- AI-Powered Love Note Generator Tool: Generates personalized romantic messages based on user input to provide content for the Love Notes Cards.

## Style Guidelines:

- Primary color: Soft pastel pink (#FFB6C1) for a romantic and gentle feel.
- Background color: Light pink to peach gradient (#FFE4E1 to #FFDAB9) for a warm and inviting atmosphere.
- Accent color: Subtle red (#FF4D6D) to highlight interactive elements.
- Headlines font: 'Great Vibes', a serif for romantic headings.
- Body font: 'Poppins', a sans-serif for a contemporary, precise feel.
- Use heart icons and subtle sparkles to enhance the romantic theme.
- Mobile-first responsive design with smooth scrolling for an elegant user experience.
- Floating heart particles animation in the background.  Use CSS animations and Intersection Observer for other animations.
